# Typer — Application Overview

A betting game for a group of friends during major football tournaments (World Cup, Euro Cup). Players predict exact match scores each match day and compete on a leaderboard.

## Tournament Structure

The tournament is divided into **8 rounds**:

1. Group stage — 3 rounds (2 matches from each tournament group)
2. Round of 32 (1/16)
3. Round of 16 (1/8)
4. Quarter-finals (1/4)
5. Semi-finals (1/2)
6. Final

## Betting Rules

- Players submit **exact score predictions** before each match day
- Deadline: start of the **first match of the day** — no exceptions, no changes after
- Predictions can be edited freely until the deadline
- In knockout rounds, only **90 minutes** count (no extra time or penalties)
- Players also predict the **Tournament Winner** (Team) and **Top Scorer** (Player)

## Bonus / Joker

Each stage has one **joker/bonus** (match of the round), chosen per player — 7 total across the tournament (3 group + 4 knockout). The joker doubles the points earned for that match.

## Scoring

| Result                                          | Points |
| ----------------------------------------------- | ------ |
| Exact score correct                             | 3 pts  |
| Correct outcome (win/draw/loss) but wrong score | 1 pt   |
| Tournament Winner correct                       | 7 pts  |
| Top Scorer correct                              | 5 pts  |

**Knockout stage multipliers** applied to match points (including joker doubling):

- Round of 32: x1.2
- Round of 16: ×1.4
- Quarter-finals: ×1.6
- Semi-finals: ×1.8
- Final: ×2.0

**Top Scorer rule:** if the top scorer finishes with fewer than 3 goals, no points are awarded in that category. If multiple players tie for top scorer, all are counted as winners.

## Tiebreaker Order

1. Total points
2. Number of exact scores predicted
3. Correct Tournament Winner
4. Correct Top Scorer
5. If still tied — multiple winners declared
